//////////////////////////このソフトについて//////////////////////////

このソフトは、Gnuplot 4.4.3にWindows CE用のコードを追加し、
ARM7以上を搭載したWindows CE 5.0以上で動作するようにしたものです。
Pocket PCやWindows Phone規格を満たしていないWindows CEでも動作します。

///////////////////////////////操作方法///////////////////////////////

同梱の"wgnuplot.exe"と"wgnuplot.mnu"を同じフォルダにコピーし、
Windows CEデバイス上でコピー先の"wgnuplot.exe"を実行してください。
メニューを日本語化するにはあらかじめコピー先の"wgnuplot.mnu"を同梱の
"wgnuplot-ja.mnu"で上書きコピーして置き換えてください。
ウィンドウの操作方法についてはGnuplotの公式サイトを参照してください。

http://gnuplot.sourceforge.net/

操作方法は基本的にWindows版Gnuplot 4.4.3と同じですが、
Windows CEの制限により次の機能は使用できません。

現在のディレクトリ
出力先の設定
プリンタへ
出力デバイスの設定
DOS 窓を開く
ヘルプ
ダンプ
PDFのサポート
GIF、PNGおよびJPEGのサポート

///////////////////////////////動作環境///////////////////////////////

OS		Windows Embedded CE 6.0
		Windows CE 5.0
CPU		ARM7 200MHz以上を推奨
メインメモリ	4MB以上の空き
ディスプレイ	640x480以上の解像度を推奨
